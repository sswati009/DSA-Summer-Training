# Assignment-02 : String Utility Toolkit

## Objective
To perform various string operations using C.

## Problem Statement
Develop a menu-driven program to perform string operations such as finding length, reversing a string, checking palindrome, counting vowels, digits, spaces, and displaying character frequency.

## Algorithm

1. Read a string.
2. Display menu.
3. Perform selected operation.
4. Repeat until Exit.

## Time Complexity

Length : O(n)

Reverse : O(n)

Palindrome : O(n)

Count Characters : O(n)

Frequency : O(n)

## Space Complexity

O(1)

## Sample Input

madam123

## Sample Output

Length = 8

Reverse = 321madam

Palindrome = Not Palindrome

Vowels = 2

Digits = 3

Spaces = 0

Character Frequency

m = 2

a = 2

d = 1

1 = 1

2 = 1

3 = 1
