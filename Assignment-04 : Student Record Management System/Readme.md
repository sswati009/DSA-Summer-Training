# Assignment-04 : Student Record Management System

## Objective

To develop a Student Record Management System using structures in C.

## Problem Statement

Create a menu-driven application that can:

- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student

using arrays of structures.

## Algorithm

1. Create a structure Student.
2. Display menu.
3. Perform selected operation.
4. Repeat until Exit.

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Display | O(n) |
| Search | O(n) |
| Update | O(n) |
| Delete | O(n) |

## Space Complexity

O(n)

## Sample Input

1

101

Rahul

89

## Sample Output

Student Added Successfully.

Roll No : 101

Name : Rahul

Marks : 89
