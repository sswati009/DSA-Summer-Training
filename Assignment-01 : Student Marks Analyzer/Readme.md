# Assignment 01 - Student Marks Analyzer

## Objective
To develop a menu-driven program for storing and analyzing student marks.

## Problem Statement
Create a C program that stores marks of N students. The program should display marks, calculate highest, lowest, total, average, search marks, and sort marks in ascending order.

## Algorithm

1. Read number of students.
2. Store marks in an array.
3. Display menu.
4. Perform selected operation.
5. Repeat until Exit.

## Time Complexity

Display : O(n)

Highest : O(n)

Lowest : O(n)

Search : O(n)

Sort : O(n²)

Average : O(n)

## Space Complexity

O(n)

## Sample Input

Number of Students

5

Marks

80 65 90 72 88

## Sample Output

Highest Marks = 90

Lowest Marks = 65

Average = 79.00

Sorted Marks

65 72 80 88 90

- Searching
